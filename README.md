# 🧩 Selectel — Frontend Test (Angular 19)

🌐 **Live Demo:**  
https://gaminv.github.io/testovoe_selectel/

Мини-приложение на **Angular 19+**, реализующее две страницы: приветствие и страницу выбора разделов с чекбоксами и автоматическим подсчётом суммы выбранных пунктов.

---

## ✨ Функциональность

- **/** — приветственная страница
- **/selector** — страница выбора раздела (Тип 1 / Тип 2)
- Отображение:
  - текущего выбранного раздела
  - количества выбранных пунктов
  - общей суммы выбранных значений
- Данные меню заданы статичным JSON в `src/app/data/menu.data.ts`

---

## 🧰 Технологии

- Angular 19+
- TypeScript
- HTML / CSS
- Angular Router
- GitHub Actions (автоматический деплой)

## ⚡ Быстрый запуск

### 1️⃣ Установите Node.js

Angular-проект требует Node.js для установки зависимостей и запуска dev-сервера.

Скачайте Node.js (версия 18 или выше):  
https://nodejs.org/

После установки проверьте версии:

```bash
node -v
npm -v
```

---

### 2️⃣ Установите зависимости

```bash
npm install
```

---

### 3️⃣ Запустите проект

```bash
npm start
```

Приложение будет доступно по адресу:

http://localhost:4200

---
