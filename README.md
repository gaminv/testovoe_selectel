# Тестовое задание — Frontend (Selectel)

Мини-приложение на **Angular 19+**, TypeScript, HTML/CSS.

## Требования

- Node.js 18+
- npm 9+

## Запуск

```bash
npm install
npm start
```

Приложение откроется по адресу: http://localhost:4200

## Сборка

```bash
npm run build
```

Артефакты появятся в папке `dist/selectel-frontend-test`.

## Структура

- **Первая страница** (`/`) — приветствие и ссылка на страницу выбора.
- **Вторая страница** (`/selector`) — меню разделов (Тип 1, Тип 2), список пунктов с чекбоксами и значениями. В шапке отображаются: текущий раздел, количество выбранных пунктов и их сумма.

Данные меню заданы статичным JSON в `src/app/data/menu.data.ts`.
